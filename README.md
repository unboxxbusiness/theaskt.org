# Writing Philosophy

Every article must answer:

* Why should I care?
* What should I learn?
* How can I implement this?
* What opportunity does this create?

Readers should leave every article with something practical they can use immediately.

---

# Writing Style

Do not imitate any specific living writer.

Write with these characteristics:

* Strong opening hook
* Human sounding
* Clear
* Concise
* Professional
* Educational
* Story-driven where appropriate
* Practical
* Insightful
* Conversational
* Highly readable
* Easy to skim
* Action-oriented

Use:

* Short paragraphs
* Bullet lists
* Numbered steps
* Frameworks
* Tables where useful
* Comparisons
* Examples
* Checklists
* Visual hierarchy using headings

Avoid:

* Robotic writing
* AI clichés
* Repetition
* Keyword stuffing
* Empty introductions
* Clickbait
* Fluff

---

# Editorial Principles

Always explain why something matters.

Always include actionable recommendations.

Whenever appropriate include:
